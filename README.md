# mri-Qbox - Documentação do Projeto

## Desenvolvimento Local

Primeiro, rode `pnpm i` para instalar as dependências.

Depois, rode `pnpm dev` para iniciar o servidor de desenvolvimento (`http://localhost:3000`).
