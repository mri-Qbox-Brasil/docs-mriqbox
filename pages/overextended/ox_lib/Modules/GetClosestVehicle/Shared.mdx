# Shared

## lib.getClosestVehicle

Retorna o identificador (handle) e as coordenadas do veículo mais próximo de um conjunto de coordenadas.

```lua
lib.getClosestVehicle(coords, maxDistance, includePlayerVehicle)
```

- coords: `vector3`
  - Coordenadas a partir das quais será feita a verificação.
- maxDistance?: `number`
  - Distância máxima para verificação.
  - Padrão: `2.0`
- includePlayerVehicle?: `boolean`
  - Se deve incluir o veículo atual do jogador. Ignorado no servidor.
  - Padrão: `false`

Retorno:

- vehicle?: `number`
- vehicleCoords?: `vector3`
