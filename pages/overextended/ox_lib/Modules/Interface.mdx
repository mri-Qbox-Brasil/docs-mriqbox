# Interface

Se você deseja alterar a **cor primária** da interface para melhor combinar com o tema do seu servidor, isso pode ser feito facilmente utilizando as convars disponíveis.

Não é necessário recompilar a UI apenas reinicie o recurso após fazer a alteração.

Convars:

```yaml
setr ox:primaryColor blue
setr ox:primaryShade 8
```

Alterar a **cor primária** mudará a cor em elementos como:

- Barra/círculo de progresso  
- Verificação de habilidade (skill check)  
- Botão central do menu radial e efeitos de hover  
- Botões de confirmação em diálogos  
- Destaque em campos de entrada  
- E diversos outros componentes visuais

Você pode encontrar a lista completa de cores e tons predefinidos neste link:

🔗 [https://v6.mantine.dev/theming/colors/#default-colors](https://v6.mantine.dev/theming/colors/#default-colors)

Se você deseja criar sua **própria paleta de cores**, é recomendável seguir o guia disponível na mesma página.

⚠️ Importante: criar uma paleta personalizada **exigirá** que você baixe o código-fonte e **compile manualmente** a interface.
