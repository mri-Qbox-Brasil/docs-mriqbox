# Shared

## lib.getNearbyPeds

Retorna o handle do ped e as coordenadas de todos os peds dentro do alcance a partir de um conjunto de coordenadas.

```lua
lib.getNearbyPeds(coords, maxDistance)
```

- coords: `vector3`
  - Coordenadas a partir das quais será feita a verificação.
- maxDistance?: `number`
  - Distância máxima para verificação.
  - Padrão: `2.0`

Retorno:

- peds: `{ ped: number, coords: vector3 }[]`
